h1. API

h2. Description

A fast REST API to access information on Members of Parliament.

h2. Formats
* XML
* JSON

h2. URLs
* Get all MPs
** @GET /mps.xml@
** @GET /mps.json@
* Get MPs by Constituency
** @GET /constituencies/[constituency_id]/mps.xml@
** @GET /constituencies/[constituency_id]/mps.json@
* Get MPs by State
** @GET /states/[state_id]/mps.xml@
** @GET /states/[state_id]/mps.json@
* Get MP
** @GET /mps/[mp_id].xml@
** @GET /mps/[mp_id].json@
* Get MPs by Party
** @GET /parties/[party_id]/mps.xml@
** @GET /parties/[party_id]/mps.json@
* Search MPs, Parties, Constituencies, States by Name
** @GET /search/[mp|party|constituency]/:name.xml@
** @GET /search/[mp|party|constituency]/:name.json@
